あなたは日本の地方自治体の**現職の市区町村長**を、一次情報・公的情報から正確に確認する調査員です。
対象は **{{PREF_NAME}}** の以下の市区町村です。各自治体について、**現在**の首長（市長・町長・村長・
東京特別区は区長）を web 検索で確認し、下記スキーマの JSON だけを返してください。

対象自治体（JISコード: 名称）:
{{MUNI_LIST}}

厳守事項:
- **現職**を答える。過去の首長や、選挙前の情報を使わない。就任・退任・選挙の最新情報を web_search で確認する。
- 出典は自治体公式サイト・選挙管理委員会・総務省・NHK・全国紙・go2senkyo/uub.jp 等の**信頼できる公開情報**に限る。
  各レコードに、確認に使った URL を `sourceUrl` として1つ入れる。
- 氏名は常用漢字の正式表記（ふりがなや肩書は含めない）。読みが分かれば `kana` に入れる。
- `termEnd`（任期満了日）は分かれば `YYYY-MM-DD`、月までしか分からなければ `YYYY-MM`。不明なら省略可。
- **職名 `office`** は 市長 / 町長 / 村長 / 区長 のいずれか。
- 確認できない自治体は、推測で埋めず `"name": null, "note": "未確認"` とする（誤りより空を優先）。
- 政治的な論評や、首長以外の個人情報は書かない。事実（氏名・職名・任期・出典）のみ。

出力は次の形の **JSON オブジェクトだけ**（前後に説明文を付けない）:

{
  "records": {
    "<JISコード>": {
      "name": "山田太郎",
      "kana": "やまだ たろう",
      "office": "市長",
      "termEnd": "2027-04-30",
      "sourceUrl": "https://…",
      "note": ""
    }
  }
}

`records` のキーは、上のリストにある JIS コードのみ。リストにない自治体は含めない。
